### Sistema de Gerenciamento de Biblioteca



Um sistema simples e eficiente de gerenciamento de livros desenvolvido em Python, focado na aplicação de conceitos de Orientação a Objetos (POO).





### Funcionalidades



* Cadastro de Livros: Adiciona novos títulos ao acervo com autor e título.
* Listagem de Acervo: Exibe todos os livros cadastrados e seus status (Disponível/Emprestado).
* Sistema de Empréstimo: Altera o status do livro e valida se ele já está emprestado antes de liberar.
* Interface via Console: Menu interativo para fácil navegação.





### Tecnologias e Conceitos Aplicados



* Linguagem: Python 3
* Paradigma: Orientação a Objetos (Classes, Métodos, Atributos)
* Estruturas de Dados: Listas (list) e manipulação de Strings
* Lógica: Estruturas de repetição (while, for) e condicional (if/else)
* Boas Práticas: Uso de Docstrings e bloco \_\_main\_\_.





### Como executar



* Certifique-se de ter o Python instalado.
* Clone o repositório ou baixe o arquivo main.py.
* Abra o terminal na pasta do arquivo.
* Execute o comando:



```python main.py```





### Exemplo de Uso



Ao iniciar o programa, o menu abaixo será exibido:


```
--- Menu da Biblioteca ---

1. Listar todos os livros
2. Adicionar um novo livro
3. Emprestar um livro
4. Sair
```
